Sparkle Vusial Novel Engine

Ru:

Дополнение для игрового движка Unity.  Он предназначен для создания визуальных новелл на базе Unity. 
Все фоны, спрайты и т.д. хранить в папке Resources в соотвествующих папках.
Сценарий должен быть в формате .txt.
dialoge_ Имя персонажа | Фраза - Это строка диалога
audio play: название песни - Воиспродить аудио файл
audio stop - Остановить воисприведение
video play: название видео - Играть видео
sprite: имя файла расположение(left,center,right) - Показать спрайт
spritehide: имя файла расположение - Спрятать спрайт
label - название - Создать лейбл
jump: название лейбла - Перейти к лейблу
choise: Текст - действие, Текст - действие... - Выбор(не более 7ми)
return - Закончить игру, возрат в главное меню
Всё писать без лишних пробелов, и не оставлять лишних строк
Пример:
label-start
dialoge_ Максим | Да
return
Функции, которые имеються:
 - Воиспровидение видео, аудио фаулов
 - Использование выборов
 - Пропуск и откат назад сюжета
